# Claude Code Skills

A collection of skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Prerequisites

Some skills include Python scripts that require [uv](https://docs.astral.sh/uv/getting-started/installation/) to run:

```bash
# macOS/Linux
curl -LsSf https://astral.sh/uv/install.sh | sh
```

## Installation

Copy skill folders to one of these locations:

| Location | Scope |
|----------|-------|
| `~/.claude/skills/` | Global (all projects) |
| `<project>/.claude/skills/` | Project-specific |
